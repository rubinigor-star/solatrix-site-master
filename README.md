# Roof Check by Solatrix

Premium mobile-first React/Vite prototype for Roof Check by Solatrix.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to GitHub Pages
1. Create a new GitHub repository.
2. Upload/push this folder.
3. Run `npm run build`.
4. In GitHub: Settings → Pages → deploy from GitHub Actions or from `/dist`.
5. Later connect a custom domain such as `roof.solatrix.energy`.
